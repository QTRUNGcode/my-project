Some basic knowledge in HTML

HTML document

All HTML documents must start with a document type declaration: <!DOCTYPE html>.
The HTML document itself begins with <html> and ends with </html>.
The visible part of the HTML document is between <body> and </body>.

If you want to attach an URL in your page, you can use <a href="put the url you want into here"> ←You can access the link when you click here </a>

Example
I want to attach the FB URL. I use
<a href="https://www.facebook.com/"> Way to facebook </a>
Then I get: → Way to facebook

HTML images

HTML images are defined with the <img> tag.
The source file (src), alternative text (alt), width, and height are provided as attributes

If you want to attach an image in your page, you can use <img scr="image address" alt="Image description">
You can adjust the image size by adding "width" and "height" into < >
